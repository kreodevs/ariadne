# analyze (UI)

Componentes reutilizables para el panel de análisis en **RepoChat** y **ProjectChat**.

- **AnalyzeScopeFields** — Prefijos de inclusión, globs de exclusión y checkbox de duplicados cross-boundary.
- **AnalyzeReportMetaBadges** — Etiquetas a partir de `reportMeta` (caché, alcance, huella degradada, capa CALL).

La construcción del objeto `ChatScope` a partir de las textareas vive en `src/utils/analyze-scope-form.ts`.
